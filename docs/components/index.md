:::tip 🤔 Components?
`Components` 将讲解 `UI` 中提供的组件。  

<p> 你可以将这些组件应用到自己的 <t-tag>App</t-tag> 中，提高开发效率，腾出更多的时间做其他的事情⏰</p>

<p> 或者作为学习的实例之一，提高自己对 <t-tag>Vue组件</t-tag> 的理解😁 </p>
:::

## 方向 :thinking:
从方向上，`Components` 将主要围绕组件的两类内容

1. 组件的 **使用**
2. 组件的 **注意事项**


<br />
<br />
<br />
<br />


## 构成 :monocle_face:

从构成上，`Components` 将主要有以下五类组件

1. **base** 基础
2. **form** 表单
3. **feedback** 反馈
4. **show** 展示
5. **navigation** 导航
6. **others** 其他
