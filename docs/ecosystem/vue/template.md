## 帝莎 :t-rex:

### [tov-template](https://gitee.com/dishait/tov-template)

一个 `vite + vue3 + ts` 开箱即用现代开发模板。

#### 特点

1. Vite 的
2. Vue3 的
3. 文件路由
4. 布局系统
5.  Mock 支持
6. Api 自动引入
7. 组件自动引入
8. 图标自动引入
9.  VueUse 支持
10. TypeScript 的
11. Windi CSS 的
12. 暗黑模式支持
13. SWR 请求支持
14. pinia 状态管理
15. pnpm 包管理器
16. 跳转进度条支持
17. Inspect 调试支持
18. 插件自动加载支持
19. Vitest 单元测试支持
20. 支持 Markdown 渲染
